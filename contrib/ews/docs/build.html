Building EWS

To build EWS use maven.
  • Download and install maven
  • Check out EWS code from the ws-axis/contrib/ews
  • cd to the project dir and type one of the following command to build the project. maven or maven jar
  • maven –o Used for offline build (you need to have the jar’s at the repository to do the offline build. But if you are build it in some machine copy the .maven dir and put it in the right place in the machine you want to build in than use –o option. But be careful for the outdated files.)
  • if you are behind a firewall create a build.properties file in the home dir with following entries

    • maven.proxy.host = cache.mrt.ac.lk (put your proxy)
    maven.proxy.port = 3128 (put your port)
    To run the EWS look at the User Guide
