Ws4J2ee tool

using with WSDL

you need the webservice.xml file, jaxrpcmapper.xml file, x.wsdl file

look at the org.apache.geronimo.ews.ws4j2ee.GenerateTest

using without WSDL

you need to have the webservice.xml file AND (SEI,Implemntation class,ejb-jar.xml file) OR ( SEI, J2SE class and web.xml file (not yet supported))

look at the org.apache.geronimo.ews.ws4j2ee.GenerateWithoutWSDLTest
