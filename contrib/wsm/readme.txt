This early JSR-181 implementation for Axis requires

Ant 1.6.1
Axis 1.2 Alpha (and ideally the latest one)
JWSDP 1.3
Tiger b32c (a.k.a J2SE 1.5.0 beta 1)

First, configure your own build.properties from build.properties.sample.
Next, set JAVA_HOME to your Tiger home.
Then, simply run "ant" :-)
