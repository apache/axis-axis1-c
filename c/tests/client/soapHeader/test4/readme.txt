This client application can be used to test the set/get/remove  soap headers with complex internal structure.
In the first part of the program it sets some soap headders with complex internal structure.Then it calls echostring method and then it gets the soap headers and remove them.

Steps to Run the Test
=====================
1) Build the source in "CHECKOUT_HOME\ws-axis\c\tests\client\soapHeader\vc" then the exe files will be created at "CHECKOUT_HOME\ws-axis\c\tests\client\soapHeader".

2) Run the client and you can verify the result by looking at the TCP Monitor.
