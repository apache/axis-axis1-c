This client application can be used to test adding/getting multiple attributes & namespaces to SOAP Headers.
In the first part of the program it sets some soap headers with the Attributes and namespaces. Then it calls echostring method.

Steps to Run the Test
=====================

Linux
=====
1) Run the client and you can verify the result by looking at the TCP Monitor.
